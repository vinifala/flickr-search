# flickr-search
This is a simple page that allows the user to input tags to search flickr's public feed.

## instructions
Run `npm install` to install all dependencies

## commands
* `npm run start` starts the dev server at `localhost:8080` and opens a browser window to that location
* `npm test` runs test suite
* `npm run lint` runs the linter (eslint)
* `npm run prettier` runs the code formater (prettier)
